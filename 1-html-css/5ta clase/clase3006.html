USO DE DISPLAY FLEX

Display flex lo que hace es que tanto lo elemento de line como los de bloque quede en linea, pero si se especifica un ancho predeterminado, en este caso a la clase caja, este se pierde ya que display flex ajutas sus elementos hijos en base al ancho original de la clase, en este caso principal.

ANCHO 200px, aca se reduce y no esta tomando como coresponde el ancho

Una forma de solucionar eso es que a la clase que tiene display flex se le coloque flex-wrap:wrap (simula un salto de linea como la etiqueta br). El flex-wrap hace que la caja quede con el ancho predeterminado y si se pasa del ancho de la caja padre automaticamente baja los elementos, como en el ekemplo de abajo.
